# 🧠 Smart Human Stress Analyzer

## 📌 Overview

This project is a Smart Stress Detection System that analyzes a user’s stress level using questionnaire data and basic behavioral inputs. It helps in identifying stress early and improving mental well-being.

## 🎯 Objective

* Detect human stress levels
* Analyze user responses
* Provide quick stress insights

## 📥 Inputs

* Questionnaire data (PSS-10 / custom questions)
* User mood input
* Optional: sleep and activity data

## ⚙️ Process

1. Collect user responses
2. Preprocess the data
3. Apply machine learning model
4. Classify stress level (Low / Medium / High)

## 📊 Output

* Stress level prediction
* Basic insights or suggestions

## 🛠️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Flask / Streamlit (optional)

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the project:
   python app.py
